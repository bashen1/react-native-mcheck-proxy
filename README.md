# react-native-mcheck-proxy

检测是否挂了代理服务器

## Installation

```sh
npm install react-native-mcheck-proxy
```

## Usage

```js
import McheckProxy from "react-native-mcheck-proxy";

// ...

const result = await McheckProxy.multiply(3, 7);
```
